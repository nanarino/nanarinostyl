## `<ul>`

既可以用于列表也可以用于子项 `ul>li.☘-list` 或 `ul.☘-list>li`

-  (R U R') U'  (R U R') 
-  (U R U' R’) 2次公式 
-  (U R U' R’) 3次公式 

```markdown
-  (R U R') U'  (R U R') 
-  (U R U' R’) 2次公式 
-  (U R U' R’) 3次公式 
```

`<ul>` 有个别名 `<menu>`


## `<ol>`

既可以用于列表也可以用于子项 `ol>li.☘-list` 或 `ol.☘-list>li`

1.  R U R' U R U2 R' 
2.  (R' F R F') R U2' R' U (R U2' R') 

```markdown
1.  R U R' U R U2 R' 
2.  (R' F R F') R U2' R' U (R U2' R') 
```
