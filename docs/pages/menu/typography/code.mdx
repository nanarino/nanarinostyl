# code

主要用来拓展markdown。但不提供语法高亮

`pre.☘-pre>code.☘-code` 里 可以用来显示多行代码

```pug
pre.☘-pre
    code.☘-code
        |    print("helloworld")
```
- `--background-color-pre` 代码块背景色RGB值
- `--background-color-pre-dark` 暗黑模式代码块背景色RGB值

为了能被 *shiki* 等库设置语法高亮

以下全局变量对 `pre.☘-pre>code.☘-code *` 生效

- `--color-code` 高亮RGB值
- `--color-code-dark` 暗黑模式高亮RGB值


## tag

<T tag='span' class='☘-tag'>#户外</T> &nbsp; <T tag='span' class='☘-tag'>#助眠</T>

```pug
span.☘-tag #户外
&nbsp;
span.☘-tag #助眠
```

相比 `code.☘-code` 除了字体更亮，样式没有区别


### kbd

<T tag='kbd' class='☘-kbd'>Ctrl</T> + <T tag='kbd' class='☘-kbd'>S</T>

```pug
kbd.☘-kbd Ctrl
| + 
kbd.☘-kbd S
```

相比 `.☘-tag` 多了一个下方阴影


## custom style

<Grid inline dataset={{ primary : "success" }}>
    <T tag='span' class='☘-tag' style="border-style:solid">#户外</T>
    <T tag='span' class='☘-tag' style="border-style:solid">#助眠</T>
</Grid>

```pug
div(data-primary="success")
    span.☘-tag(style="border-style:solid") #户外
    span.☘-tag(style="border-style:solid") #助眠
```

***

- 全局变量 `--background-color-[tag|code]` 背景颜色RGB值
- 全局变量 `--color-[tag|code]` 字体和边框颜色RGB值

    '--color-code': `var(--magenta-1)`,
