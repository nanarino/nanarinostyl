## `data-size`

`{i}` 值范围 `1`到 `6` : i级标题的大小和边距

<T class="☘-title" data-size='3'>𓀉 𓀊 𓀋 𓀌 𓀍</T>
<T class="☘-title" data-size='4'>𓀘 𓀙 𓀚 𓀛 𓀜 𓀝 𓀞𓀀</T>
<T class="☘-title" data-size='5'>𓀁 𓀂 𓀄 𓀅 𓀆 𓀇 𓀈</T>
<T class="☘-title" data-size='6'>𓀁 𓀂 𓀃 𓀄 𓀅 𓀆 𓀇 𓀈</T>
<T>𓀎 𓀏 𓀐 𓀑 𓀒𓀓  𓀖 𓀗</T>

```pug
.☘-title(data-size=3) 𓀉 𓀊 𓀋 𓀌 𓀍
.☘-title(data-size=4) 𓀘 𓀙 𓀚 𓀛 𓀜 𓀝 𓀞𓀀
.☘-title(data-size=5) 𓀁 𓀂 𓀄 𓀅 𓀆 𓀇 𓀈
.☘-title(data-size=6) 𓀁 𓀂 𓀃 𓀄 𓀅 𓀆 𓀇 𓀈
.☘-title 𓀎 𓀏 𓀐 𓀑 𓀒𓀓  𓀖 𓀗
```

## subtitle

副标题字体大小比主标题低一级

<T
    class="☘-title"
    data-size='1'
> 红楼梦 <T tag="span" class="☘-title-subtitle">戚序本</T> </T>

```pug
.☘-title(data-size=1)
    |红楼梦
    span.☘-title-subtitle 戚序本
```

***

<T
    class="☘-title"
    tag='h2'
> 红楼梦 <T tag="span" class="☘-title-subtitle" style="float:right">戚序本</T> </T>

```pug
h2.☘-title
    |红楼梦
    span.☘-title-subtitle(style={ float: 'right' }) 戚序本
```

## `data-has-prefix`

<T class="☘-title" tag='h6' data-has-prefix>石头记</T>
<T class="☘-title" tag='h6' dataset={{
    'has-prefix': true,
    'primary': "danger"
}}>石头记</T>
<T
    class="☘-title"
    sub='戚序本'
    dataset={{
        'has-prefix': true,
        "size": 3
    }}
    style="--color-title-prefix: var(--gold-5)"
> 红楼梦 <T tag="span" class="☘-title-subtitle">戚序本</T> </T>

全局变量 `--color-title-prefix`、`--color-blockquote-prefix` 可以自定义前缀主题颜色RGB值

```pug
h6.☘-title(data-has-prefix) 石头记
h6.☘-title(data-has-prefix data-primary='danger') 石头记
.☘-title(data-has-prefix data-size=3 style={
    '--color-title-prefix': 'var(--gold-5)'
})
    |红楼梦
    span.☘-title-subtitle 戚序本
```
