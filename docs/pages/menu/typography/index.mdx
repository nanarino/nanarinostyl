# typography

用来拓展markdown（`.mdx`）的样式一般会被归为此类中，但链接 `<a>` 没有被纳入

## font

- `.☘-font-serif` 衬线字体
- `.☘-font-mono` 等宽字体
- `.☘-connot-select` 无法选中文字

和颜色一样，字体大小和行高有10个级别。

| lv | font-size | line-height |   alias   |
| -- | --------- | ----------- | --------- |
|  1 |      12px |        20px | base      |
|  2 |      14px |        22px | body / h6 |
|  3 |      16px |        24px |        h5 |
|  4 |      20px |        28px |        h4 |
|  5 |      24px |        32px |        h3 |
|  6 |      30px |        38px |        h2 |
|  7 |      38px |        46px |        h1 |
|  8 |      46px |        54px |           |
|  9 |      56px |        64px |           |
| 10 |      68px |        76px |           |

以下是全局变量

- `--font-family-base`
- `--font-family-serif`
- `--font-family-mono`
- `--line-height-body` （部分组件有覆写）
- `--font-size-body` （部分组件有覆写）

可以像下面这样写以改变全局变量使所有标签默认大小变成3级（即 `<h5/>` 的大小）

```sass
body
    --font-size-body: var(--font-size-3)
    --line-height-body: var(--line-height-3)
```

但标题其不受影响，需要自行选择 `<h1/>` 到 `<h6/>` ，对应markdown里的 `#` 数量。
