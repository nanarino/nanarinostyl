# panel

这个样式设计来自于 [_ui5 panel_](https://sap.github.io/ui5-webcomponents/components/Panel/)

<T style={{
    'border': '2px solid rgb(var(--black))',
    'border-radius': 'var(--border-radius-lg)',
    'overflow': 'hidden',
}}>
    <T tag="details" class="☘-panel">
        <summary>进入</summary>
        <T>到顶了</T>
    </T>
</T>

```pug
div(style={
    'border': '2px solid rgb(var(--black))',
    'border-radius': 'var(--border-radius-lg)',
    'overflow': 'hidden',
})
    details.☘-panel
        summary 进入
        div 到顶了
```


## `data-primary`

`details.☘-panel[data-primary]` 这个属性需要启用后才能继承主题和拥有某些局部样式

<T 
    tag="details" 
    class="☘-panel"
    data-primary
>
    <summary>进入</summary>
    <T>到顶了</T>
</T>

```pug
details.☘-panel(data-primary)
    summary 进入
    div 到顶了
```

## `data-marker=">"`

预设了另一个箭头符号<Icon style="vertical-align:sub" name="direction/outline/right" width={16} height={16} />，使用 `data-marker=">"` 启用，代替默认的<Icon style="vertical-align:sub" name="direction/outline/caret-right" width={16} height={16} />

<T 
    tag="details" 
    class="☘-panel"
    data-marker=">"
    data-primary="warning"
    style="color:rgb(var(--gold-8))"
>
    <summary>进入</summary>
    <T>到顶了</T>
</T>

```pug
details.☘-panel(
    data-marker=">"
    data-primary="warning"
    style="color:rgb(var(--gold-8))"
)
    summary 进入
    div 到顶了
```

## custom style

全局变量 `--color-panel` 可以覆盖面板背景色RGB值

marker箭头由 `summary::before` 的 `mask` 遮罩图片实现，可以自定义添加你需要的<Icon style="vertical-align:sub" name="direction/outline/caret-right" width={16} height={16} />图形

<T
    tag="details"
    class="☘-panel"
    style={{
        color: 'rgb(var(--blue-6))',
        '--color-panel': 'var(--blue-1)'
    }}
    data-primary
    data-marker="(>)"
>
    <summary>
        <span>&nbsp;正在安装</span>
        <span>🥰</span>
    </summary>
    <T>
        <T tag="p"
            class="☘-paragraph"
            data-size="2"
            style="margin:0"
        >😍😍😍😍😍😍<span style="float:right">99%</span></T>
    </T>
</T>

```pug
details.☘-panel(
    data-marker="(>)"
    data-primary
    style={
        color: 'rgb(var(--blue-6))',
        '--color-panel': 'var(--blue-1)'
    }
)
    summary 
        span 正在安装
        span 🥰
    div
        p.☘-paragraph(data-size=2 style="margin:0") 
            | 😍😍😍😍😍😍
            span(style="float:right") 99%

style(lang="stylus").
    .☘-panel[data-marker="(>)"] summary::before
        -webkit-mask url("docs/icons/direction/outline/right-circle.svg")
        mask @-webkit-mask
        background-color rgb(var(--blue-6))
```
