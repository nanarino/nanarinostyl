## custom style

- 全局变量 `--color-badge` 可以自定义圆点和波纹颜色RGB值
- 全局变量 `--badge-processing-weight` 调整 波浪大小
- 全局变量 `--indent-badge` 调整文字左内边距，原本是用 `text-indent` 属性实现，但是在Safari中使用 `::before` 的元素不支持 `text-indent`。如果使用 `text-indent` 效果见下：

<T tag="span" class="☘-badge" data-processing data-primary="danger" style="
    --indent-badge: 0; 
    text-indent: 1.2em; 
    font-size: 20px; 
    --badge-processing-weight: 2px;
    --color-badge: var(--primary-7)
">`debug`</T>

```pug
span.☘-badge(data-processing data-primary="danger" style={
    "--indent-badge": "0",
    "text-indent": "1.2em",
    "font-size": "20px",
    "--badge-processing-weight": "2px",
    "--color-badge": "var(--primary-7)"
})
    span.☘-code.☘-font-mono debug
```

上述示例代码依然使用 `text-indent`，在Safari中可以观察到波纹是椭圆形，且圆心也偏移了。

***

- 全局变量 `--margin-badge` 圆点的边距，可以用来反向调整圆点大小

使用 `<svg/>` 需要设置容器或者其自身宽高。

<T
    tag="span"
    class="☘-tag"
    data-primary="danger"
    style="
        --indent-badge: 30px;
        --margin-badge: 6px;
        border-style: dashed
    "
>
    <T tag="span" class="☘-badge" data-processing>
        <Icon
            name="general/outline/bug"
            role="img"
            aria-label="bug"
            width="30"
            style="vertical-align: middle"
        />
    </T>
</T>

```pug
span.☘-tag(data-primary="danger" style={
    "--indent-badge": "30px",
    "--margin-badge": "6px",
    "border-style": "dashed"
})
    span.☘-badge(data-processing)
        svg(
            role="img"
            aria-label="bug"
            width="30"
            height="30"
            style="vertical-align: middle"
        )
```
