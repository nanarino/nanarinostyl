# paragraph

markdown中的空格另起一行即为 `<p>`

<T class="☘-paragraph">西游记</T>

```pug
.☘-paragraph 西游记
```


## `data-size`

`{i}` 值范围 `1`到 `6` : 同i级标题的大小和边距

<T class="☘-paragraph" data-size="1">诗曰：</T>

```pug
.☘-paragraph(data-size=1) 诗曰：
```

## `data-ellipsis`

`{i}` 值范围 `1`到 `6` : 超出i行会变成省略号

<T class="☘-paragraph" dataset={{
    "size": 2,
    "ellipsis": 2
}}>混沌未分天地乱，茫茫渺渺无人见。
自从盘古破鸿蒙，开辟从兹清浊辨。
覆载群生仰至仁，发明万物接为善。
欲知造化会元功，须看西游释厄传。</T>

```pug
.☘-paragraph(data-size=2 data-ellipsis=2)
    |混沌未分天地乱，茫茫渺渺无人见。
    |自从盘古破鸿蒙，开辟从兹清浊辨。
    |覆载群生仰至仁，发明万物接为善。
    |欲知造化会元功，须看西游释厄传。
```
