# divider

以下是拓展后的markdown的 `<hr>` , 可以使用 `---` 或 `***`

***

## `<div>`

<T class="☘-divider" />

```pug
.☘-divider
```

## `<span>`

<p>楚河<T tag="span" class="☘-divider" />漢界</p>

```pug
p
    |楚河
    span.☘-divider
    |漢界
```

## `<hr>`

<T tag="hr" class="☘-divider" />

```pug
hr.☘-divider
```

## custom style

<T tag="hr" class="☘-divider" style="border: 1px dotted rgb(var(--primary-2)"/>

```pug
hr.☘-divider(style="border: 1px dotted rgb(var(--primary-2)")
```
