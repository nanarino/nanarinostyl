## `data-primary`

设置 `textarea.☘-textarea[data-primary]` 可以覆盖全局css变量 `--outline-color-focus`

直接使用局部变量 `--color-textarea` 也可以设置主题颜色RGB值

<Grid inline>
{
    ['primary', 'success', 'danger', 'warning'].map(primary => 
        <T
            tag="textarea"
            class="☘-textarea"
            title={primary}
            placeholder={primary}
            data-primary={primary}
        />
    )
}
<T
    tag="textarea"
    class="☘-textarea"
    title="custom"
    placeholder="custom"
    style="--color-textarea: var(--blue-6)"
/>
</Grid>

```pug
each primary in ['primary', 'success', 'danger', 'warning']
    textarea.☘-textarea(
        title=primary
        placeholder=primary
        data-primary=primary
    )
textarea.☘-textarea(
    title="custom"
    placeholder="primary"
    style="--color-textarea: var(--blue-6)"
)
```

## `.sm` & `.lg`

不设置宽高时默认是2行高20个等宽半角字符宽

全局变量 `--padding-textarea` 设置内边距

有下面两个预设尺寸：

- `textarea.☘-textarea.sm`
    <br/><T tag="textarea" class="☘-textarea sm" title="尺寸更小的文本域" placeholder="更小" />
- `textarea.☘-textarea.lg`
    <br/><T tag="textarea" class="☘-textarea lg" title="尺寸更大的文本域" placeholder="更大" />
